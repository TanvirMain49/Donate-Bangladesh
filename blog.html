What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

How do you select an element from the DOM?

In order to interact with the document object model, one can select an element using methods such as getElementById, getElementsByClassName, getElementsByTagName, querySelector, or querySelectorAll. These methods search and change particular tags in the webpage considering their ids, classes, tags and so on.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. It allows us to attach a single event listener for elements that exist now or in the future

How do you manipulate an element's attributes and styles using the DOM?

Through accessing an element in the DOM, it is possible to change its attributes and styles by the use of several methods such as setAttribute(), getAttribute() and removeAttribute() for attributes, or even style such as CSS. Concerning styles, the style property over the element is adjusted for the direct change of the CSS properties
